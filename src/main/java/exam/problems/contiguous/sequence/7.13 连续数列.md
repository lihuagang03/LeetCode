

# 7.13 连续数列
> https://pintia.cn/problem-sets/2004497238921478144/exam/problems/type/7?problemSetProblemId=2004497238955032588

[16.17. 连续数列](https://leetcode.cn/problems/contiguous-sequence-lcci/description/)

给定一个整数数组，找出总和最大的连续数列，并返回总和。


进阶：如果你已经实现复杂度为 O(n) 的解法，尝试使用更为精妙的分治法求解。


### 输入格式:
数组nums

### 输出格式:
连续子数组的最大和

### 输入样例:
在这里给出一组输入。例如：
```
-2,1,-3,4,-1,2,1,-5,4
```
### 输出样例:
在这里给出相应的输出。例如：
```
6
```
解释： 连续子数组 [4,-1,2,1] 的和最大，为 6。
